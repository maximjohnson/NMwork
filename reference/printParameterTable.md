# Print parameter table to console, flextable, latex, or (latex) pdf file

Print parameter table to console, flextable, latex, or (latex) pdf file

## Usage

``` r
printParameterTable(
  pars,
  engine = "kable",
  format,
  footnotes = NULL,
  script = NULL,
  file.mod,
  include,
  include.pattern,
  drop,
  drop.pattern,
  include.fix,
  caption
)
```

## Arguments

- pars:

  A parameter table as generated by \`NMwork::createParameterTable()\`.

- include:

  A character vector of symbol values to include. All else will be
  dropped.

- include.pattern:

  A regular expression run on the \`symbol\` column. What matches will
  be kept.

- drop:

  A character vector of symbol values to drop. All else will be kept.

- drop.pattern:

  A regular expression run on the \`symbol\` column. What matches will
  be dropped.

- include.fix:

  Either a logical or "NotZero". \`TRUE\` means fixed parameters are not
  filtered out, \`FALSE\` means they are filtered out, "NotZero" means
  they are only kept if not equal to zero.
