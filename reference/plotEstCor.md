# Plot correlation matrix heatmap from completed NONMEM run

Plot correlation matrix heatmap from completed NONMEM run

## Usage

``` r
plotEstCor(file.lst, pars = NULL, label.by = "parameter", col.label = NULL)
```

## Arguments

- file.lst:

  \`.lst\` file from completed NONMEM run

- pars:

  a table of parameter names and labels to use for renaming parameters
  to descriptive labels
